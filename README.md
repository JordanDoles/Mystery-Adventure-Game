# Mystery Investigation Game
Authors: Jordan Doles and Emma Nei

## Overview
This project is a simple GUI-based mystery game built using Python and Tkinter.  
The player investigates different rooms, collects clues, and is able to accuse suspects to determine the culprit and complete the game. 

## Description 
A valuable family heirloom has been stolen from a safe in the living room.  
The player must explore the house, examine objects, and question suspects to determine who is responsible.

## Requirements
- Python version 3.14
- tkinter (usually included in Python installation)
- Pandas

## How to Run
1. Navigate to the project folder
2. Run file: main.py

## Features
- Room navigation (e.g., kitchen, living room)
- Searching rooms to collect clues
- Suspect questioning with alibis
- Accusation system to choose the culprit
